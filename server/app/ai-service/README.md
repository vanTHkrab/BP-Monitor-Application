# AI Service

This directory contains the implementation of the AI Service, which is responsible for handling AI-related functionalities in the application. The AI Service provides APIs for various AI tasks, such as natural language processing, image recognition, and more.

## Installation with uv
To install the AI Service, follow these steps:
1. Navigate to the `ai-service` directory:
   ```bash
    cd server/app/ai-service
    ```
2. Install the dependencies using `uv`:
   ```bash
    uv install
    ```
3. Start the AI Service:
   ```bash
    uv run fastapi dev main.py
    ```
